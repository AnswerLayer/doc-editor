AnswerLayer Architecture

Version: 1.0
Date: 2026-03-18

AnswerLayer is a BYOC (Bring Your Own Cloud) application — all infrastructure runs inside the customer's AWS account. Deployment is orchestrated by Nuon, which provisions a runner agent via CloudFormation that executes Terraform to manage the application stack (ECS, RDS, ALB, S3).
